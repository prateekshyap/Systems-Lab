==============
INSTRUCTIONS TO EXECUTE THE CODE
==============

1. All the programs are written in JAVA.

2. Class files are provided. The following commands can be entered in command prompt/terminal to execute the code.

Question-1 -> java Restaurant
Question-2 -> java Wedding
Question-3 -> java Restaurant

3. To recompile the codes, go inside respective folder and

   1) type "make" and press enter in Linux terminal.
   2) type "javac *.java" and press enter in windows Command Prompt.

4. Input and Output file names can be given dynamically during runtime.
